# MsZohaCollage
MsZohaCollage Student Information and other information gathering tools.

### Installation :

```bash
pkg update && pkg upgrade -y
pkg install git -y
pkg install python -y
git clone https://github.com/ShTasrif/MsZohaCollage
pip install requests
cd MsZohaCollage
python mszoha.py
```
### Run Tools

```bash
mszoha
```
